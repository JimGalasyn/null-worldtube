# Figures — Paper 1

All figures are generated by `generate_figures.py`. To regenerate a specific figure:

```bash
cd papers/figures
python3 generate_figures.py 1    # Figure 1: torus knot multi-panel
python3 generate_figures.py 2a   # Figure 2a: predicted vs measured masses
python3 generate_figures.py 2b   # Figure 2b: predicted vs measured dimensionless
python3 generate_figures.py 3    # Figure 3: derivation chain
python3 generate_figures.py 4    # Figure 4: Koide generation sphere
python3 generate_figures.py 5    # Figure 5: torus EM modes
python3 generate_figures.py 6    # Figure 6: Pythagorean resonance
python3 generate_figures.py all  # All figures
```

## Figure 1: Torus Knot Representations

**File:** `figure1_torus_knot.png` | Full-width (`figure*`)

Four-panel (2x2) figure showing how the NWT framework represents different particle types:
- **(a)** Electron: (2,1) knot on torus with R, r annotations and circulation arrows
- **(b)** K± meson: phase defect on fatter torus (r/R=0.5), visible gap and color discontinuity
- **(c)** Proton: (3,4) mode on thinner torus (r/R=1/3), exact Pythagorean closure 3²+4²=5²
- **Cross-section**: Meridional slice showing R, r, and alpha=r/R

## Figure 2a: Predicted vs Measured Masses

**File:** `figure2a_masses.png` | Single-column

Log-log scatter of 11 mass and energy-scale predictions spanning five orders of magnitude. Inset: electroweak cluster resolved. Bottom: error histogram (median 0.4%).

## Figure 2b: Predicted vs Measured Dimensionless Parameters

**File:** `figure2b_dimensionless.png` | Single-column

Linear scatter of 10 dimensionless parameters (mixing angles, CP phases, coupling constants). Inset: small-value cluster resolved. Bottom: error histogram (median 1.0%).

## Figure 3: Derivation Chain

**File:** `figure3_derivation_chain.png` | Single-column

Flowchart showing the logical derivation chain from (p,q,k) + m_e to all 23 Standard Model parameters.

## Figure 4: Koide Generation Sphere

**File:** `figure4_koide_sphere.png` | Single-column

Three mass eigenstates sqrt(m_i) as unit vectors on a sphere with 120-degree azimuthal separation (Z_3 symmetry). Koide angle theta_K determines polar angles.

## Figure 5: Electromagnetic Modes on the Torus

**File:** `figure5_torus_modes.png` | Single-column

Grid of 14 modes (13 after zero-mode subtraction): 4 toroidal gamma, 1 poloidal gamma, 4 mixed gamma, 1 metric gamma, 3 metric weak (W+, W-, Z), 1 zero mode (g_0). Summary: sin^2(theta_W) = 3/13 = 0.231.

## Figure 6: Pythagorean Resonance

**File:** `figure6_pythagorean_resonance.png` | Full-width (`figure*`)

Two-panel figure showing the resonance condition (kp)^2 + q^2 = N^2 on the unwrapped torus:
- **(a)** Meson sector (k=2): K± mode falls between integer circles, phase defect delta=1
- **(b)** Baryon sector (k=3): proton mode sits exactly on N=5 circle, delta=0 (stable)
